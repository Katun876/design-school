Живая обратная связь
Финал — кейс в портфолио
Профессия UI/UX изнутри. Осваиваем Figma: фреймы, слои, автолэйаут, компоненты, стили. Первый прототип.
Законы визуального восприятия, шрифтовая шкала, цветовые системы, сетки. Воссоздаём реальный интерфейс с нуля.
User interviews, анкетирование, конкурентный анализ. Переводим инсайты в дизайн-решения.
Информационная архитектура, навигация, lo-fi прототипы для быстрой проверки идей.
Атомарный дизайн, компонентная библиотека в Figma. Строим полноценную дизайн-систему с нуля.
Интерактивные прототипы, переходы, motion-принципы. Как добавить жизнь без лишней сложности.
Тестируем прототипы с реальными пользователями, анализируем, приоритизируем правки.
Mobile-first, breakpoints, адаптация компонентов. Делаем мобильное приложение для портфолио.
Передача макетов разработчикам, документация, Inspect. Говорим с командой на одном языке.
Самостоятельный проект от ресёрча до макетов. Оформляем кейс с контекстом, проблематикой, аргументацией.
